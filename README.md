# 🏥 منصة راسخ لنمذجة المشاريع الصحية

## 📋 نظرة عامة

**راسخ** هي منصة رقمية متكاملة لنمذجة وإدارة المشاريع الصحية، تهدف إلى توحيد المعايير وتسهيل تكرار المبادرات الناجحة في مختلف المناطق. تم تطويرها خصيصاً لجمعية الكوثر الصحية.

## ✨ المميزات الرئيسية

### 🎯 مكتبة النمذجة
- أدلة تشغيلية مفصلة لكافة جوانب المشاريع
- نماذج جاهزة للاستخدام المباشر
- توثيق شامل لكل مرحلة تنفيذية

### 📊 لوحة القيادة
- متابعة حية لتقدم المهام
- إحصائيات مفصلة عن الإنجاز
- عرض المراحل والمهام بشكل تفاعلي

### 👥 إدارة الفريق
- هيكل تنظيمي واضح
- تحديد المسؤوليات والمهام
- متابعة أداء أعضاء الفريق

### 🗺️ تصميم الموقع ورحلات الزوار
- كروكي تفاعلي للموقع
- تخطيط مسارات الحركة
- رحلات الزوار المختلفة (المستفيد، المتدرب، الزائر الرسمي، الضيف الثقافي)

### 📚 الدليل التشغيلي
- دليل شامل قابل للطباعة
- تصدير PDF احترافي
- محتوى منظم بفصول وأقسام

### 📝 مكتبة النماذج
- نماذج تشغيلية جاهزة
- وثائق معيارية موحدة
- سهولة التحميل والاستخدام

## 🚀 المشاريع المتاحة

### مشروع صوم وصحة
مبادرة صحية ميدانية متكاملة في شهر رمضان تقدم:
- خدمات طبية شاملة
- برامج تدريبية متخصصة
- أنشطة توعوية وتثقيفية
- استقبال الزيارات الرسمية

## 🛠️ التقنيات المستخدمة

- **Frontend:** React 18 + TypeScript
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion
- **Icons:** Lucide React
- **Build Tool:** Vite
- **State Management:** React Hooks

## 📦 التثبيت والتشغيل

```bash
# تثبيت المكتبات
npm install

# تشغيل المشروع في وضع التطوير
npm run dev

# بناء المشروع للإنتاج
npm run build

# معاينة البناء
npm run preview
```

## 🎨 المكونات الرئيسية

- **RasikhLanding**: الصفحة الرئيسية للمنصة
- **RasikhLibrary**: مكتبة المشاريع
- **DashboardView**: لوحة القيادة
- **PhasesView**: عرض المراحل والمهام
- **TeamStructureView**: الهيكل التنظيمي
- **ProjectBookView**: الدليل التشغيلي
- **JourneysView**: تصميم الموقع ورحلات الزوار
- **TemplatesView**: مكتبة النماذج
